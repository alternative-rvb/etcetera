# 🎙️ Etcetera

Application de dictée vocale gratuite pour Windows.
Basée sur Whisper (OpenAI) — 100% locale, aucune donnée envoyée sur internet.

> *"...et cetera"* — parce que la parole ne s'arrête jamais vraiment.

## Télécharger le .exe

1. Va dans l'onglet **Actions** de ce repo GitHub
2. Clique sur le dernier workflow "Build Etcetera.exe"
3. Télécharge **Etcetera-Windows** dans la section Artifacts

## Utilisation

- **Bouton 🔴** : démarrer/arrêter la dictée
- **Ctrl+Shift+Espace** : maintenir pour dicter, relâcher pour injecter
- Fonctionne dans VS Code, navigateur, Word, Notepad...

## Langues supportées

Français, English, Español, Deutsch, Italiano, Português
